## **Question 1: Understanding Docker Images**

---

### Objective
Run a Docker container using the `python:3.13` image, enter it with a Bash shell, and identify the installed `pip` version.

### Steps

- Verify Docker is running
```bash
docker --version
```

- Pull the python image
```bash
docker pull python:3.13
```

- Run the container with Bash as entrypoint
```bash
docker run -it --entrypoint bash python:3.13
```

- Check the pip version
```bash
pip --version
```

- Exit the container
```bash
exit
```

### pip version is `pip 25.3`
